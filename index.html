Spot the Difference: The Game

One Memorizer studies the room, then looks away while the Editors secretly change it — or disguise themselves as furniture. The Memorizer returns with limited ammo: wrong accusations burn a shot, and if they run dry the Editors win. Editors win if anyone survives the round. Get caught and you're out.
